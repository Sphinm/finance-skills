# Volume-Price Signal Library — 量价信号库

A 股特有的量价模式识别。用于判断入场时机、持有信心、离场信号。

---

## 健康趋势信号（持有/加仓）

### 1. 缩量新高

- **形态**：股价创近期新高，但成交量/换手率低于前一次高点
- **含义**：卖方惜售，买方不需要太多量就能推高价格 = 共识一致看多
- **换手率阈值**：<3% 创新高 = 极度健康
- **操作**：持有不动，或等回踩 5MA 后加仓
- **案例**：沪电股份 2026.5.18 涨1.23%，换手仅2.56%

### 2. 缩量回踩5/10日线

- **形态**：上涨后回调 3-5%，成交量缩至高点的 40-50%，价格触及 5/10 日均线后企稳
- **含义**：获利盘轻微兑现后无人继续卖 = 浮筹洗干净了
- **操作**：回踩不破均线 + 次日止跌 → 买入信号
- **止损**：跌破 10 日线且放量 → 认错出局

### 3. 板块联动启动

- **形态**：同赛道 3 只以上个股同日异动（涨幅>3%），板块净流入>10亿
- **含义**：板块共识正在形成，资金系统性进入
- **操作**：从异动个股中选 PEG 最低的作为建仓标的
- **案例**：2026.5.18 生益涨停 + 沪电/深南同涨 + 元件板块净流入42亿

### 4. 底部放量不跌

- **形态**：股价处于低位（近 20 日低点附近），突然成交量放大 2-3 倍，但股价不跌反而微涨或持平
- **含义**：有大资金在低位吸筹，卖单被全部接住
- **操作**：连续 2-3 天出现 → 开始关注；出现缩量阳线突破 → 确认买入

## 危险信号（减仓/清仓）

### 5. 高换手涨停

- **形态**：涨停板当日换手率>15%（大盘股>10%）
- **含义**：上方卖压极大，买方硬扛住但随时可能开板
- **对比**：低换手涨停（<5%）= 封板坚决，大概率次日继续
- **操作**：高换手涨停不追，等次日观察是否开板回落

### 6. 天量长上影

- **形态**：单日成交额达到近 5 日均量的 3 倍以上 + 留长上影线（上影>实体的 50%）
- **含义**：冲高遇到巨大卖压，大资金在高位出货
- **操作**：当日减仓 1/3-1/2；次日如果低开 → 清仓

### 7. 放量滞涨

- **形态**：成交量连续 2-3 天放大（>1.5倍日均），但股价涨幅<1%
- **含义**：有人在出货，但出货量太大导致价格"顶住了"
- **操作**：开始设紧止损（前低-2%），一旦破位立即离场

### 8. 主力出 + 散户进

- **形态**：主力资金（超大单+大单）净流出 + 中小单净流入
- **含义**：机构在分发筹码，散户在接盘 = 顶部分发
- **数据源**：东方财富个股资金流向 / 同花顺 Level-2
- **操作**：连续 2 天出现 → 减仓 50%；连续 3 天 → 清仓
- **案例**：中科曙光 2026.5.12 主力-6.37亿，散户+3.95亿

### 9. 利好不涨

- **形态**：明确利好出来（业绩超预期/大合同/政策利好），盘中冲高但收盘涨幅<1%
- **含义**：有人趁利好出货 = "利好出尽"
- **操作**：立即提高警惕；如果次日低开 → 减仓
- **案例**：恒生互联网 2026.5.14 阿里腾讯Q1业绩利好后盘中+3.77%，收盘仅+0.47%

## 入场触发器

### Trigger A: 回踩均线买入

```
条件：
  ① 近期有明确上涨趋势（20日内涨幅>15%）
  ② 回调至5日线或10日线
  ③ 回调日成交量 < 上涨日成交量的50%
  ④ 不破前一个重要低点
  ⑤ 所在板块当日未出现净流出>5亿

操作：买入1/3目标仓位
止损：跌破10日线且放量（换手>5%）
```

### Trigger B: 突破前高买入

```
条件：
  ① 股价突破近20日最高价
  ② 突破日成交量>前5日均量的2倍（量比>2）
  ③ 板块同日净流入
  ④ 突破后不回补缺口（30分钟内不跌回突破价以下）

操作：买入1/3目标仓位
止损：跌回突破价以下（即"假突破"确认）
```

### Trigger C: 确认加仓

```
条件：
  ① 已持仓且浮盈>5%
  ② 股价沿5日线缩量上行
  ③ 板块持续净流入（连续3日）
  ④ 无明显利空催化（无解禁/减持/行业政策风险）

操作：加仓至2/3目标仓位
止损：整体持仓成本-8%
```

## 离场触发器

### Exit 1: 止损

```
条件（满足任一）：
  - 跌破买入价-8%（绝对止损）
  - 跌破10日线且放量（趋势破坏）
  - 板块连续3日净流出>10亿（beta反转）

操作：全部清仓，不犹豫
```

### Exit 2: 止盈

```
条件（满足任一）：
  - 达到目标价（通常基于PEG=1或板块PE中位数）
  - 出现天量长上影（信号6）
  - 出现主力出+散户进连续2天（信号8）

操作：
  - 达目标价 → 减仓50%，剩余设trailing stop
  - 危险信号 → 减仓50-100%
```

### Exit 3: 换仓

```
条件：
  - 当前持仓板块beta明显弱于新兴主线
  - 持仓标的PEG已>1.5（估值到位）
  - 新标的PEG<0.8且通过四层漏斗

操作：不等回本，直接换到期望值更高的标的
```

## A股特有模式

### 涨停板次日策略

| 涨停类型 | 次日大概率走势 | 操作 |
|----------|----------------|------|
| 低换手封板（<5%）+ 基本面催化 | 高开继续涨 | 可以开盘后15分钟内进 |
| 低换手封板 + 纯情绪（无催化） | 冲高回落 | 观望 |
| 高换手封板（>15%） | 分歧加大，可能大幅波动 | 不参与 |
| 一字板（换手<1%） | 惜售极端，但开板时获利盘巨大 | 不追一字板，等开板后观察 |

### 北向资金信号

| 信号 | 含义 |
|------|------|
| 连续5日净买入同一板块 | 外资系统性看好 → 跟随 |
| 单日净买入>100亿 | 重大事件驱动，通常是底部信号 |
| 连续5日净卖出 | 外资撤退，可能有你不知道的利空 |

### 融资余额信号

| 信号 | 含义 |
|------|------|
| 融资余额持续上升 + 股价上涨 | 杠杆资金看好，趋势强化 |
| 融资余额高位 + 股价滞涨 | 杠杆资金被套，强平风险 |
| 融资余额快速下降 | 杠杆资金清仓，短期利空出尽 |

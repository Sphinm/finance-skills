# 亨通光电 光纤涨价+AI需求 Case Study (May 2026)

亨通光电作为全球第三大光纤光缆供应商，受益于光纤价格从25元→80元/芯公里的3倍涨价（AI数据中心互联需求驱动）。Q1利润+98.5%，PE 40x，PEG仅0.41 = 当前AI算力链中估值最被低估的标的之一。

## Decision

**结论**: BUY（核心仓位候选）

**理由**:
1. PEG 0.41 是当前讨论的所有标的中**最低的**
2. 光纤涨价周期确认（25→80-105元/芯公里），且产能扩张有限→涨价可持续
3. 利润增速98.5% vs PE 40x = 市场严重低估其增速持续性
4. 海缆+海洋通信提供额外增量（非AI依赖的第二增长曲线）
5. 四层漏斗全部通过

**操作计划**:
- 介入方式: 回踩5日线建仓
  - 第一批(1/3): 回踩68-70元区间
  - 第二批(1/3): 确认趋势（缩量沿5MA上行3天后）
  - 第三批(1/3): 放量突破75元
- 止损位: ¥65（跌破10日线且放量）
- 目标位: ¥90-100（对应PE 50-55x，仍低于板块高估值标的）
- 仓位: 15-20%

---

## Lessons

1. **涨价周期是最好的增速保障**：不需要预测"下个季度会不会好"，价格从25涨到80就是答案
2. **PEG 0.4级别在正常市场中极为罕见**：要么市场还没发现，要么有隐藏风险。亨通的情况是前者（光纤涨价是2026Q1才开始的新信号）
3. **双轮驱动比单一逻辑更安全**：即使AI需求不及预期，海缆/海洋通信订单也能支撑增长
4. **利润翻倍+PE仅40x = 市场在"等确认"**：一旦Q2继续验证，估值重估空间巨大

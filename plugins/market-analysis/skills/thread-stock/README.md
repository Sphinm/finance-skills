# Thread Stock

A-share / HK / ADR thematic stock selection — four-layer funnel, 9 pitfalls, case studies.

## Triggers

- A股选股、板块分析、资金流向、主线判断
- AI算力链、供应链拆解、PEG筛选、量价分析

## Setup

1. Install `cn-market-analysis` plugin (or symlink this skill).
2. Pair with `cn-data-providers` for live data and `cn-screening-tools` for scan scripts.

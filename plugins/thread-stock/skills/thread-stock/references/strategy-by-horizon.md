# Strategy by Horizon — 长中短线分层策略

三种持仓周期，三套筛选逻辑，三种仓位管理方式。脚本通过 `--horizon` 参数切换。

---

## 策略总览

| 维度 | 短线 (swing) | 中线 (position) | 长线 (trend) |
|------|-------------|-----------------|-------------|
| **持仓周期** | 3-5 天 | 2 周 - 3 个月 | 3 个月 - 1 年 |
| **核心驱动** | 量价动量 + 资金异动 | 财报验证 + 均线回踩 | 产业周期 + 估值底部 |
| **分析权重** | tape 60% > flow 25% > 基本面 15% | 基本面 40% > tape 30% > flow 30% | 基本面 60% > 估值 25% > tape 15% |
| **仓位上限** | 单票 10%，总短线 ≤30% | 单票 20-25% | 单票 25-30% |
| **止损线** | -5%（硬止损） | -8% 或破 MA10 放量 | -12% 或破 MA60 |
| **止盈方式** | +8-15% 或放量滞涨即走 | 目标价到位减半 + trailing | 季报不及预期 或 PEG>2 |
| **适合标的** | 板块龙头/首板连板/异动放量 | 四层漏斗全过 + 回踩到位 | Tier 1-2 + PEG<0.5 + 多季度验证 |

---

## 短线策略 (Swing / 3-5 天)

### 核心理念

> 不关心公司值多少钱，只关心短期内**谁在买、量有多大、趋势能持续几天**。

### 筛选条件

| 条件 | 阈值 | 原因 |
|------|------|------|
| 当日板块净流入 | Top 5 板块内的个股 | 顺板块 beta |
| 个股量比 | >1.5（突然放量） | 资金异动信号 |
| 5 日涨幅 | +5% ~ +15% | 有动量但未 parabolic |
| RSI(5) | 50-75 | 有动量但未到极端 |
| 换手率 | 3-10% | 活跃但不是游资对倒 |
| 非游资快进快出 | 龙虎榜无知名游资大额卖出 | 避免被当接盘侠 |
| 流通市值 | >100 亿 | 流动性保障 |

### 入场触发

| Trigger | 条件 | 操作 |
|---------|------|------|
| **放量突破** | 突破前 5 日高点 + 量比>2 + 板块同日净流入 | 突破确认后进 1/2 仓 |
| **涨停次日** | 低换手封板(<5%) + 有基本面催化 + 次日高开不超3% | 开盘 15 分钟内进 1/2 仓 |
| **V 型反转** | 盘中跌>5% 后 V 型拉回收阳 + 量比>1.5 | 尾盘进 1/2 仓 |

### 出场规则

| 条件 | 操作 |
|------|------|
| 亏 -5% | 无条件止损 |
| 盈 +8-15% | 减仓 2/3，剩余设当日低点-2% trailing |
| 第 3 天未涨（横盘） | 减仓 1/2（动量衰减） |
| 第 5 天 | 无论盈亏清仓（纪律） |
| 天量长上影 | 当日清仓 |

### 脚本筛选逻辑

```python
# 短线候选条件
def is_swing_candidate(r):
    return (
        r["vol_ratio"] >= 1.5          # 放量
        and 5 <= r["chg5"] <= 15       # 有动量
        and r["rsi"] < 75              # 未极端超买
        and r["rsi"] > 50              # 有动量方向
    )
```

## 中线策略 (Position / 2 周 - 3 个月)

### 核心理念

> **四层漏斗全过 + 回踩均线** = 用确定性换时间，让经营杠杆的利润弹性慢慢兑现。

### 筛选条件

这就是现有的四层漏斗，沿用不变：

| Layer | 条件 |
|-------|------|
| 1. Supply Chain | AI 翻倍→订单翻倍？物理不可替代 |
| 2. Earnings | 扣非增速>30%，经营杠杆比>1.0，毛利率改善 |
| 3. Valuation | PEG<1.0（同板块内） |
| 4. Flow | RSI 40-65 + 距 MA20 在 -5%~+5% + 缩量回踩 |

### 入场触发

| Trigger | 条件 | 操作 |
|---------|------|------|
| **A（回踩买入）** | 上涨趋势中回调至 MA10-MA20 + 缩量 + 不破前低 | 买入 1/3 |
| **B（突破买入）** | 放量突破前高 + 板块同日净流入 | 买入 1/3 |
| **C（确认加仓）** | 浮盈>5% + 缩量沿 MA5 上行 + 板块连续 3 日净流入 | 加至 2/3 |

### 出场规则

| 条件 | 操作 |
|------|------|
| 亏 -8% 或跌破 MA10 放量 | 减仓 50% |
| 亏 -12% 或跌破 MA20 | 清仓 |
| 盈利到目标价（PEG=1.5 对应价格） | 减仓 50%，trailing |
| 出现放量滞涨连续 2 天 | 减仓 50% |
| 出现主力出+散户进连续 3 天 | 清仓 |
| 季报不及预期（经营杠杆比<1） | 清仓换标的 |

### 脚本筛选逻辑

```python
# 中线候选条件（现有逻辑）
def is_position_candidate(r):
    return (
        r.get("peg") and r["peg"] < 1.0
        and 40 < r["rsi"] < 65
        and r["dev_ma20"] < 10
        and r.get("leverage") and r["leverage"] > 1.0
    )
```

## 长线策略 (Trend / 3 个月 - 1 年)

### 核心理念

> 买**产业周期的确定性** + **估值的安全边际**，不频繁操作，赚整个景气上行周期的钱。

### 筛选条件

| 条件 | 阈值 | 原因 |
|------|------|------|
| 产业链位置 | Tier 1 或 Tier 2 | 只买最上游、最不可替代的 |
| 扣非利润增速 | >30%，且连续 2 个季度以上 | 排除一次性因素 |
| 经营杠杆比 | >1.5 | 利润弹性确认 |
| PEG | **<0.5** | 严格低估才进 |
| 利润率 | >10% | 排除薄利的系统集成商 |
| 均线排列 | MA20 > MA60（中期上升趋势确认） | 不左侧抄底 |
| 行业景气度 | 下游 capex 上调 / 产品涨价 / 产能满载 | 周期向上验证 |

### 入场触发

长线不追求精确择时，更关注"区间"而非"点位"：

| 触发 | 条件 | 操作 |
|------|------|------|
| **首次建仓** | PEG<0.5 + MA20>MA60 + 季报验证 | 建仓 1/3 |
| **加仓 1** | 持仓浮盈>10% + 下一季报持续验证 | 加至 2/3 |
| **加仓 2** | 持仓浮盈>20% + 行业景气度加速 | 加至满仓 |
| **底部区域** | PEG<0.3 + RSI<35 + 板块超卖 | 逆势建仓 1/4（严格止损） |

### 出场规则

| 条件 | 操作 |
|------|------|
| 亏 -12% | 减仓 50% |
| 跌破 MA60 且放量 | 清仓（中期趋势破坏） |
| 季报经营杠杆比跌破 1.0 | 减仓 50%（景气度拐点） |
| 连续 2 个季度利润增速下滑 | 清仓（周期见顶信号） |
| PEG 升至 >2.0 | 减仓到 1/3（估值到位） |
| 达到市场一致目标价 | 减仓 50%（别贪） |

### 脚本筛选逻辑

```python
# 长线候选条件
def is_trend_candidate(r):
    return (
        r.get("peg") and r["peg"] < 0.5
        and r.get("profit_margin") and r["profit_margin"] > 0.10
        and r.get("leverage") and r["leverage"] > 1.5
        and r["tier"].startswith(("Tier1", "Tier2"))
    )
```

## 仓位配置原则

```
总仓位 = 短线仓（≤30%）+ 中线仓（≤40%）+ 长线仓（≤30%）

短线: 3-5 只 × 5-10% = 最多 30%
中线: 2-3 只 × 10-20% = 最多 40%
长线: 1-2 只 × 15-30% = 最多 30%

现金: 始终保留 ≥10%（应对突发机会/止损补充）
```

### 仓位互转规则

| 场景 | 操作 |
|------|------|
| 短线标的季报超预期，通过四层漏斗 | 升级为中线持仓 |
| 中线标的连续 3 个季度经营杠杆>2 | 升级为长线持仓 |
| 长线标的季报失速（杠杆<1） | 降级为中线，设更紧止损 |
| 中线标的动量消失（横盘>2周） | 降级为短线止损纪律 或 清仓 |

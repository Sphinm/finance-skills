# Weekly Scan Checklist — 周度扫描清单

每周末（周六/周日）执行，耗时约 15-20 分钟。目标：确认持仓健康、更新候选池、发现新机会。

---

## □ 板块资金流向

**数据源**: akshare `stock_fund_flow_industry` / 东方财富板块资金流向

| 检查项 | 方法 | 行动 |
|--------|------|------|
| 本周净流入 Top 10 板块 | 运行扫描脚本 | 记录，对比上周变化 |
| 主线板块资金是加速还是减速 | 本周 vs 上周净流入额 | 加速→持有；减速→收紧止损 |
| 有没有新板块连续 3 日净流入 | 每日数据汇总 | 有→研究是否有产业催化 |
| 主线板块内是否出现"价涨资金出" | 板块涨但ETF净赎回 | 有→分发信号，不追高 |

**判断标准**:
- 主线板块连续 5 日净流入 >20 亿/日 → 趋势强化
- 主线板块净流入转负 → 预警，收紧止损
- 新板块连续 3 日净流入 >10 亿/日 → 值得研究

## □ 持仓标的健康检查

对每只持仓标的逐一检查：

| 检查项 | 阈值 | 行动 |
|--------|------|------|
| RSI > 70 | 超买预警 | 设紧止损，准备减仓 |
| RSI > 80 | 严重超买 | 减仓 1/3-1/2 |
| 出现"主力出+散户进"连续 2 天 | 分发信号 | 减仓 50% |
| 出现放量滞涨（量增价不增）| 高位出货 | 设紧止损（前低-2%） |
| 出现天量长上影 | 冲高被砸 | 当日减 1/3，次日低开则清 |
| 距止损位 < 3% | 接近触发 | 准备执行，不犹豫 |
| 距止损位 > 15% | 安全 | 可以考虑上移止损至成本价 |
| 跌破 10 日线且放量 | 趋势破坏 | 减仓 50% |

**输出**: 每只持仓的健康状态（健康/预警/危险）+ 下周操作计划。

## □ 候选池更新

### 新财报超预期标的

| 检查 | 来源 | 条件 |
|------|------|------|
| 本周新发布的季报 | 东方财富/巨潮 | 在 AI_CHAIN_TICKERS 范围内 |
| 扣非利润增速 | akshare 财务指标 | >50% 且经营杠杆比 >1.5 |
| PEG | 计算 | <1.0 |

### 主线回踩标的

| 检查 | 条件 | 行动 |
|------|------|------|
| RSI 回到 40-60 | 从超买区修正 | 进入观察 |
| 回踩 MA10-MA20 | 距 MA20 在 -5%~+5% | Trigger A 候选 |
| 回踩过程缩量 | 量比 <0.8 | 加分 |
| 支撑位不破 | 不破前低/关键均线 | 确认 |

### Trigger 检查

| Trigger | 条件 | 操作 |
|---------|------|------|
| A（回踩买入） | 上涨趋势中回调至 5/10 日线 + 缩量 + 不破前低 + 板块未净流出 | 买入 1/3 |
| B（突破买入） | 放量突破前高（量比>2）+ 板块同日净流入 + 30分钟不回补 | 买入 1/3 |
| C（确认加仓） | 已持仓浮盈>5% + 缩量沿 5 日线上行 + 板块连续 3 日净流入 | 加至 2/3 |

## □ 低估值机会扫描

**目标**: 找到主线板块内 PE/PEG 健康但尚未被市场充分关注的标的。

| 筛选条件 | 阈值 |
|----------|------|
| 所属板块 | 半导体 / 元器件 / 通信设备 / 光学光电 |
| PEG | < 1.0（同板块前 30%） |
| 近 1 周涨幅 | < 板块平均涨幅的 50%（相对滞涨） |
| 扣非增速 | > 30% |
| 利润/营收增速比 | > 1.2 |
| 非游资快进快出 | 最近一周无龙虎榜游资席位大额买卖 |

**资金异动但非游资的判断方法**:
- 有异动：近 5 日某日成交量 >2 倍均量
- 非游资：龙虎榜未出现"财通杭州/东方拉萨/华鑫上海"等知名游资席位
- 机构进场：龙虎榜出现"机构专用"席位买入 → 加分

## □ 机会成本审计

| 对比 | 方法 | 行动阈值 |
|------|------|----------|
| 持仓 vs 主线最强标的 | 过去一周涨跌幅对比 | 差距 >10% → 考虑换仓 |
| 持仓 vs 候选池最优 | PEG + 技术面综合对比 | 候选明显优于持仓 → 换仓 |
| 持仓亏损标的 | 被套 >30 天且仍亏损 | 有更优标的 → 减半换仓 |

**核心问题**: "如果我现在空仓，我会以当前价格买入这只票吗？"
- Yes → 持有
- No → 卖出（无论盈亏）

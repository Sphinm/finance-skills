## Tier 1: 上游材料 — 确定性最高

"无论谁做芯片/谁做服务器，都绕不过去的物理材料。"

| 环节 | 核心标的 | 代码 | 竞争格局 | 为什么不可替代 |
|------|----------|------|----------|----------------|
| **CCL(覆铜板)** | 生益科技 | 600183 | 国内第一/全球第二, 14%市占率 | AI服务器PCB层数3-5x, 价值量8-12x, 英伟达M9唯一认证 |
| CCL(高频) | 南亚新材 | 688519 | 高频高速隐形冠军 | PTFE/LCP差异化, M6-M8批量, M9导入中 |
| **光纤光缆** | 亨通光电 | 600487 | 全球第三 | 光纤价格25→80元/芯公里, AI数据中心互联刚需 |
| 光纤光缆 | 中天科技 | 600522 | 全球前五 | 海缆+陆缆双轮驱动 |
| 石英材料 | 菲利华 | 300395 | 国内石英龙头 | 半导体扩产用石英, 但AI直接关联度低 |

## 选股优先级矩阵

```
          确定性高 ←————————————→ 确定性低
弹性大 ↑  [南亚新材]  [强一股份]   [英维克]
        |  [亨通光电]  [澜起科技]   [浪潮信息]
        |  [生益科技]  [沪电股份]   [中科曙光]
弹性小 ↓  [中天科技]  [深南电路]   [科大讯飞]
```

**首选区域**：左上角（确定性高 + 弹性大）
**次选区域**：左下角（确定性高 + 弹性小）或中上（确定性中 + 弹性大）
**回避区域**：右侧（确定性低，无论弹性大小）

## 如何发现"下一个生益科技"

### 信号扫描清单

1. **全球大厂 capex 上调**：英伟达/AMD/微软/谷歌 说"增加资本支出" → 找国内对应供应链
2. **行业涨价信号**：CCL涨价/光纤涨价/存储涨价 → 找涨价直接受益者
3. **产能扩张公告**：下游宣布扩产计划 → 找上游设备/材料供应商
4. **认证突破**：某公司获得大客户认证（英伟达/台积电/三星）→ 订单确定性飞跃
5. **出口数据异常**：某产品出口额同比暴增 → 找国内生产商

### 时间轴

```
T-6月: 全球大厂宣布capex上调 → 开始研究供应链
T-3月: 产业链公司发布业绩预告/快报 → 初步验证
T-0:   一季报/半年报发布 → 确认标的
T+1周: 机构研报上修目标价 → 市场开始price in
T+1月: 板块高潮/涨停潮 → 如果没上车，等回调
```

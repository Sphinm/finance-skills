# 英维克 液冷叙事vs报表崩塌 Case Study (May 2026)

英维克被市场冠以"液冷龙头"标签，入选英伟达NVL72供应商推荐名录。但Q1财报暴露严重问题：利润同比-82%，PE飙升至2882x，应收账款是营收的2.6倍，经营现金流-3.86亿。这是"叙事压过报表"（Pitfall 08）的教科书案例。

---

## Setup

- **标的**: 英维克 (002837)
- **分析日期**: 2026-05-18
- **股价**: ¥102.12
- **市值**: 998亿
- **PE**: 2882x（！）
- **所属板块**: 液冷/温控设备
- **板块当日表现**: 涨+1.62%

---

## Fundamental Check

| 指标 | 数据 | 达标? |
|------|------|-------|
| Q1营收增速 | +26.03% | △ (及格但不突出) |
| Q1利润增速 | **-81.97%** | ✗✗✗ |
| 毛利率变化 | -2.16pct | ✗ |
| 收现比 | 经营现金流-3.86亿 | ✗✗ |
| PEG | 2882/(-82) = N/A | ✗ (利润为负增长) |

**暴露的问题**:
- 液冷业务仅占收入7-8%，远未到放量阶段
- 信用减值损失从752万→3015万（+300%），应收账款坏账风险飙升
- 应收账款30.61亿 vs 季度营收11.75亿 = 客户欠了近3个季度的钱
- 财务费用大增（汇兑损失+利息支出）

- 今日换手率: 4.62%
- 成交额: 4006万
- 量价形态: 微涨，无明显趋势

---

## Decision

**结论**: AVOID（淘汰）

**理由**:
1. Layer 2（业绩验证）彻底失败：利润-82%，四层漏斗在第二层就被淘汰
2. PE 2882x 意味着市场为"未来可能的液冷爆发"预付了极高溢价，但当前盈利完全撑不起
3. 应收账款/坏账风险是定时炸弹
4. 同样的主题（AI散热），有更便宜+更确定的选择（如果液冷真爆发，PCB公司一样受益）

**不碰的理由清单**:
- ✗ PEG 不可计算（利润为负增长）
- ✗ 增收不增利（Pitfall 02）
- ✗ 叙事压过报表（Pitfall 08）
- ✗ 液冷业务占比<10%（Pitfall 01 — 蹭概念边界）
- ✗ 收现比极差（应收是营收2.6倍）

---

## Lessons

1. **"XX龙头"标签不能替代财报分析**：市场给的标签是滞后的，报表是实时的
2. **增收不增利是最清晰的淘汰信号**：营收+26%但利润-82% = 经营杠杆严重负向
3. **应收账款/营收比>2 = 红色警报**：说明客户不付钱或付不起钱
4. **同板块比较暴露问题**：生益科技PEG 0.58 vs 英维克PE 2882x，选择不言自明
5. **概念业务占比<20%的公司，不应按概念估值定价**

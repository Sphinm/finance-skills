# Cross-Market Positioning — 跨市场配置

A 股 vs 港股 vs 美股中概：什么时候该在哪个市场？ETF 仓位管理与"套牢陷阱"规避。

---

## 三市场特征对比

| 维度 | A 股 | 港股 | 美股中概 |
|------|------|------|----------|
| 定价权 | 内资主导 | 外资+南向资金 | 美元流动性主导 |
| 涨跌幅 | 10%/20%(科创) | 无限制 | 无限制 |
| 流动性 | 极好（万亿级日均） | 分化（龙头好/小票差） | 集中于头部 |
| 估值敏感因子 | 政策+流动性+情绪 | **美债利率** | 美联储+地缘 |
| 散户占比 | 高（60%+） | 低（<20%） | 极低 |
| 信息效率 | 偏低（情绪驱动） | 中等 | 较高 |
| T+0/做空 | 无/融券极少 | 有/做空活跃 | 有/做空活跃 |

## 市场选择决策树

```
问题1: 催化剂来自哪里？
  ├── 国内政策/产业（国产替代、内需刺激）→ A股
  ├── 全球利率/美元周期 → 港股（利率敏感）
  └── 公司自身业绩/美股联动 → 美股中概

问题2: 当前美债10Y利率趋势？
  ├── 利率上行期 → 回避港股成长股（duration被杀）
  ├── 利率见顶/下行 → 港股成长股有估值修复空间
  └── 利率平稳 → 看个股alpha

问题3: A-H溢价处于什么位置？
  ├── A股溢价极高（>40%）→ 港股更有性价比
  ├── A股溢价正常（20-30%）→ 各做各的
  └── 港股溢价（罕见）→ A股更安全
```

## 各市场最佳应用场景

### A 股首选

| 场景 | 原因 | 典型标的 |
|------|------|----------|
| 国产替代主题 | 政策驱动+内资共识快 | 中微/北方华创/强一 |
| AI算力硬件链 | 供应链公司在A股 | 生益/沪电/亨通/澜起 |
| 情绪驱动的板块轮动 | A股散户多→情绪放大→弹性大 | 存储/PCB涨停潮 |
| 次新股/科创板 | 国内独有标的 | 强一股份 |

### 港股首选

| 场景 | 原因 | 典型标的 |
|------|------|----------|
| 互联网平台（低估值周期） | 美债利率下行时弹性最大 | 腾讯/阿里/美团 |
| 利率拐点交易 | 港股对利率变化极敏感 | 恒生科技ETF |
| AH折价极端时 | 港股明显便宜于A股 | H股ETF |
| 高股息防守 | 港股红利税较A股低 | 中国移动/中海油 |

### 美股中概首选

| 场景 | 原因 | 典型标的 |
|------|------|----------|
| 公司业绩驱动 | 美股定价更理性 | PDD/BABA(ADR) |
| 美元资产配置 | 分散汇率风险 | 中概ETF(KWEB) |
| 期权交易（vol play） | 美股期权流动性远好于港股 | BABA/PDD options |

## ETF 仓位管理规则

### 建仓规则

| 规则 | 内容 |
|------|------|
| 单只ETF最大仓位 | 不超过总资产 25% |
| 单一市场方向最大暴露 | 不超过总资产 40% |
| 新建仓 | 首次进入不超过目标仓位 50%，确认后加满 |
| 加仓条件 | 持仓浮盈>5% 且趋势未变 |

### 止损规则

| 规则 | 内容 |
|------|------|
| 绝对止损 | -12% 无条件减仓 50% |
| 趋势破坏止损 | 跌破 20 日线且放量 → 减仓 50% |
| Beta 反转止损 | 基准指数连跌 3 天且每天>1% → 全部清仓 |
| 机会成本止损 | 持有 30 天仍亏损且有更优标的 → 换仓 |

### 加仓规则

| 条件 | 操作 |
|------|------|
| 持仓浮盈 + 趋势确认 | 加仓至目标 |
| 持仓亏损但触及超卖极值（RSI<25） | **不补仓**，除非有新催化 |
| 计划内的左侧定投 | 按固定金额/固定间距，不因亏损加速 |

## ETF 套牢陷阱（Pitfall 07 详解）

### 识别你是否掉入了 ETF 陷阱

| 问题 | 如果答案是"Yes" = 陷阱 |
|------|------------------------|
| 你的 ETF 连跌 >10% 且无明确反转信号？ | Yes |
| 基准指数跌破所有均线（空头排列）？ | Yes |
| ETF 连续净赎回 >5 天？ | Yes |
| 你买入的唯一理由是"已经跌了很多应该到底了"？ | Yes |
| 你有其他板块的更好机会但因为"想等回本"而不动？ | Yes |

如果 3 个以上 Yes → 你在陷阱里。

### 脱困策略

```
策略A: 减半换仓
  ① 卖出50%仓位（接受亏损）
  ② 资金转入当前主线板块最强标的
  ③ 保留50%设"加仓触发"条件

策略B: 全清重来
  ① 全部清仓（如果5个Yes都是）
  ② 冷静一周不操作
  ③ 用新的四层漏斗重新选标的

策略C: 定投摊薄（仅限以下条件全部满足）
  ① ETF估值处于历史<10%分位
  ② 你有明确的"至少持有12个月"的时间框架
  ③ 你有持续的新资金来源
  ④ 你不需要这笔钱在6个月内用
```

### 关键心理纠偏

> **"回本"不是一个有效的投资目标。**
>
> 正确的问法是："以当前价格、当前信息，如果我没有持仓，我会买入吗？"
>
> - 会 → 持有（甚至加仓）
> - 不会 → 卖出（无论是否亏损）

## 恒生互联网专题

### 核心驱动变量

恒生科技指数的走势由三个变量决定：

```
恒生科技 = f(美债10Y, 中国经济预期, 互联网公司盈利)
```

| 变量 | 权重 | 当前状态(2026.5) |
|------|------|------------------|
| 美债10Y | ~40% | 上行（CPI超预期）→ 负面 |
| 中国经济预期 | ~30% | 中性（PMI平稳） |
| 互联网盈利 | ~30% | 正面（阿里云+40%, 腾讯capex+） |

**当前判断**：三个变量中最重要的一个（利率）方向不对，即使盈利改善也难以抵消利率压制。

### 底部确认信号

恒生科技真正见底需要以下信号中至少 2 个：

1. **美债10Y见顶回落**（从当前水平下行>30bp）
2. **恒生科技RSI<25 + 连续3天缩量后出现放量大阳线**
3. **南向资金连续5日大幅净买入（>50亿/天）**
4. **VIX跌回<18 + 人民币汇率企稳**

目前 0/4 满足 → **不是抄底时机**。
